# ⏳ HOURFIX Backend

## DESPLEGAR EN RAILWAY (10 minutos)

1. Sube esta carpeta a GitHub
2. Ve a railway.app → New Project → Deploy from GitHub
3. Añade variable de entorno: JWT_SECRET=cualquier_texto_largo_secreto
4. ¡Listo! Railway te da una URL pública

## PROBAR EN LOCAL

```bash
npm install
node server.js
```

API en: http://localhost:3000
